# Personal Landing Page

This is my personal landing page built using **HTML, CSS, and responsive design**.

## Features
- Responsive design for desktop and mobile
- Hero section with heading, text, and image
- About section with multiple paragraphs
- Footer with contact links
- Wave SVG divider for smooth separation

## Tools & Technologies
- HTML5
- CSS3
- Flexbox 
- Responsive design techniques
